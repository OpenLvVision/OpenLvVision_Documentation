OpenLvVision_Overlays.lvlib:CopyOverlay.vi

Copies an image overlay from a source to a destination image. If the destination image already contains an overlay, this VI combines the source overlay with the pre-existing one.

Src Reference In

An Image reference is a handle to the actual image data in memory. It's important to note that you are responsible for manually releasing (deallocating) the memory associated with this image reference when it's no longer needed to prevent memory leaks.

Groups

An array of strings specifying the names of overlay groups to copy. If this array is empty, all existing overlay groups will be copied.
