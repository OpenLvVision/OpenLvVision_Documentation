Image Name

A unique identifier assigned to the created image reference

Image Type

Datatype of the pixel

Border

Image borders are essential for kernel-based image processing. This surrounding layer of pixels provides the necessary neighborhood context for kernels to operate correctly, especially at image edges, ensuring accurate convolution results across the entire image.

Image Reference Out

An Image reference is a handle to the actual image data in memory. It's important to note that you are responsible for manually releasing (deallocating) the memory associated with this image reference when it's no longer needed to prevent memory leaks.
