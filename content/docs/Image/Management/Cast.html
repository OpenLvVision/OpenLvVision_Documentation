Converts the image's pixel data type.

For 8-bit and 16-bit images, the scaling method for this conversion is set by the specified "Method".

Image Src Reference In

An Image reference is a handle to the actual image data in memory. It's important to note that you are responsible for manually releasing (deallocating) the memory associated with this image reference when it's no longer needed to prevent memory leaks.

Image Type

ImageFormat enum defines pixel data type:

MonoU8 (8-bit unsigned ),

MonoU16 (16-bit unsigned),

MonoI16 (16-bit signed),

MonoSGL (single-precision float),

ColorU32 (8x4 unsigned integer, RGBA),

ColorU64 (16x4 unsigned integer, RGBA).

Method

This is valid only for 16-bit to 8-bit conversions.

Specifies the casting approach when changing pixel data types

- Use Bit Depth (0): Casts based on the image's bit depth information.

- Shift (1): Casts by right-shifting the pixel values by a specified "Number of Shifts".

- Use Lookup Table (2): Casts using a provided "Lookup Table".

Number of Shifts

For 16-bit to 8-bit conversions.

This specifies the number of bits to right-shift the pixel values during casting.

Lookup Table

Array where the index is the original value, and the values at that indexs are the new values.
