PixelPointer

    		Image Border Size

    Border/padding around image data.

    		Pixel Pointer

    Address to the pixel.

    		LineWidth

    Represents the total number of pixels in a horizontal image line in memory. This value is calculated by adding the image's horizontal resolution (`xRes`), the left and right border sizes, and any left and right alignment padding.

    		PixelSize_Bytes

    The size of a single pixel in bytes for the current image format.

    		TransferMaxSize

    Maximum transferable bytes from the pixel pointer to the image end.

Image Reference In

An Image reference is a handle to the actual image data in memory. It's important to note that you are responsible for manually releasing (deallocating) the memory associated with this image reference when it's no longer needed to prevent memory leaks.
