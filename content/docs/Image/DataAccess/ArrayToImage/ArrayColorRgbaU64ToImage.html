Image Reference In

An Image reference is a handle to the actual image data in memory. It's important to note that you are responsible for manually releasing (deallocating) the memory associated with this image reference when it's no longer needed to prevent memory leaks.

Pixel

Image Pixels (U64) contains the pixel values as clusters of four unsigned 16-bit integers.

    		Color (U64)

      		Plane 3 (Blue)

      Plane 3 (Blue) is the value of the blue plane.

      		Plane 2 (Green)

      Plane 2 (Green) is the value of the green plane.

      		Plane 1 (Red)

      Plane 1 (Red) is the value of the red plane.

      		Plane 4 (alpha)

      Plane 4 (alpha) is the value of the alpha plane.
