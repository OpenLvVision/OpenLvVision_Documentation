OpenLvVision_Image.lvlib:File.lvlib:ReadImageFile.vi

Reads an image from a file (supporting BMP, TIFF, JPEG, JPEG2000, PNG) and can also read non-standard formats if the "read raw file" option is enabled.

Image Reference In

An Image reference is a handle to the actual image data in memory. It's important to note that you are responsible for manually releasing (deallocating) the memory associated with this image reference when it's no longer needed to prevent memory leaks.

File Path

Path to the image file

Load Color Pallete

Loads a color palette for remapping monochrome image colors. The palette assigns a display color to each grayscale value (index).

Use File Path Image Type

Extracts the file type from the provided path

ImageFileOptions

    		File Data Type

    indicates how the image file is encoded.

    		Offset to Data

    		Use Min Max

    Specifies whether a user-defined minimum and maximum are being used.

    		Optional Min Value

    		Optional Max Value

    		Byte Order

    Byte Order (Endianness) defines how multi-byte pixel values are stored.

    		Read Raw File

    Indicates a non-standard file format. If true, further options specify how to read the data.
