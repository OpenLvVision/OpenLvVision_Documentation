AllocType

PAGE_LOCKED: Allocates pinned memory, which is essential for accelerating data transfer between CPU and GPU.

SHARED: Specifies zero-copy memory, which enables mapping host memory to the GPU address space, reducing copy overhead, especially on integrated GPUs.

WRITE_COMBINED: Allocates write-combined memory, which is not cached by the CPU, making it efficient for write-only data destined for the GPU.
