OpenLvVision_OpenCv.lvlib:Cuda.lvlib:GpuMat.lvclass:GpuMat Download_HostMem.vi

Transfers data from the GPU device memory (GpuMat) back to a page-locked (pinned) host memory buffer (HostMem). Just like the upload function, using pinned memory accelerates the transfer. Providing a `Stream` reference allows this download to happen asynchronously.

For detailed information, please refer to the OpenCV documentation
