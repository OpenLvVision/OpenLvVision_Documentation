Object by Reference!

Creates and initializes a new CUDA Stream.

A stream represents a dedicated, asynchronous execution queue.

For detailed information, please refer to the OpenCV documentation

 error in (no error)

The error in cluster can accept error information wired from VIs previously called. Use this information to decide if any functionality should be bypassed in the event of errors from other VIs.

StreamFlags

cudaStreamDefault: Creates a standard stream. This stream will synchronize with the global Default Stream (Stream 0).

cudaStreamNonBlocking: Creates a completely independent, non-blocking stream.
