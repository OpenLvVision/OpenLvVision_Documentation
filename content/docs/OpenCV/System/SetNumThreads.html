OpenLvVision_OpenCv.lvlib:SetNumThreads.vi

OpenCV will try to set the number of threads for the next parallel region.

If threads == 0, OpenCV will disable threading optimizations and run all it's functions sequentially. Passing threads < 0 will reset threads number to system default.

For detailed information, please refer to the OpenCV documentation
