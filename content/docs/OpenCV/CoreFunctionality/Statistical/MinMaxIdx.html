OpenLvVision_OpenCv.lvlib:MinMaxIdx.vim

Finds the global minimum and maximum in an array.

The extremums are searched across the whole array or, if mask is not an empty array, in the specified array region. Multi-channel input is supported without mask and indexes.

For detailed information, please refer to the OpenCV documentation
