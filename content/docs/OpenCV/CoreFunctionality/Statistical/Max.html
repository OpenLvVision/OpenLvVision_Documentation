Calculates per-element maximum of two arrays or an array and a scalar.

The function max calculates the per-element maximum of:

  • Two arrays:
    {{< katex >}}$ \texttt{dst}(I) = \max(\texttt{src1}(I), \texttt{src2}(I)) ${{< /katex >}}
  • An array and a scalar:
    {{< katex >}}$ \texttt{dst}(I) = \max(\texttt{src1}(I), \texttt{value}) ${{< /katex >}}

For detailed information, please refer to the OpenCV documentation
