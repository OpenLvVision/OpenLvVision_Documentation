OpenLvVision_OpenCv.lvlib:FindNonZero.vim

Returns the list of locations of non-zero pixels.

The function do not work with multi-channel arrays. If you need to find non-zero elements across all the channels, reinterpret the array as single-channel. Or you may extract the particular channel using either extractImageCOI, or mixChannels, or split.

{{< alert context="info" text="If only count of non-zero array elements is important, countNonZero is helpful. If only whether there are non-zero elements is important, hasNonZero is helpful. " />}}

For detailed information, please refer to the OpenCV documentation
