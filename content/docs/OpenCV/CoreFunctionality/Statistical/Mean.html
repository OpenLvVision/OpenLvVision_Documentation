OpenLvVision_OpenCv.lvlib:Mean.vim

Calculates an average (mean) of array elements.

The function mean calculates the mean value M of array elements, independently for each channel, and returns it:

{{< katex >}}$ \begin{array}{l} N = \sum\limits_{I : \ \texttt{mask}(I) \ne 0} 1 \\ M_c = \frac{\sum\limits_{I : \ \texttt{mask}(I) \ne 0} \texttt{mtx}(I)_c}{N} \end{array} ${{< /katex >}}
