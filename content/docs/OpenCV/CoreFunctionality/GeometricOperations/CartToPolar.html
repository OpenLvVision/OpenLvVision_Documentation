OpenLvVision_OpenCv.lvlib:CartToPolar.vim

Calculates the magnitude and angle of 2D vectors.

{{< katex >}}$ \begin{aligned} \texttt{magnitude}(I) &= \sqrt{\texttt{x}(I)^2 + \texttt{y}(I)^2}, \\ \texttt{angle}(I) &= \texttt{atan2}\!\big(\texttt{y}(I),\, \texttt{x}(I)\big)\,[\cdot\, 180/\pi] \end{aligned} ${{< /katex >}}

The angles are computed with an accuracy of approximately 0.3 degrees. For the point (0,0), the angle is set to 0.

For detailed information, please refer to the OpenCV documentation
