Divides a multi-channel array into several single-channel arrays.

The function split< splits a multi-channel array into separate single-channel arrays:

{{< katex >}}$ \texttt{mv}[c](I) = \texttt{src}(I)_c ${{< /katex >}}

If you need to extract a single channel or perform more complex channel permutations, use mixChannels.

For detailed information, please refer to the OpenCV documentation
