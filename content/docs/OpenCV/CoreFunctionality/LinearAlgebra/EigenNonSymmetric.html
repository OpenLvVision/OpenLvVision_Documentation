OpenLvVision_OpenCv.lvlib:EigenNonSymmetric.vim

Calculates eigenvalues and eigenvectors of a symmetric matrix.

The function calculates eigenvalues and optionally eigenvectors of the square matrix src:

{{< katex >}}$ \texttt{src} \cdot \texttt{eigenvectors.row}(i)^T = \texttt{eigenvalues.at}(i) \cdot \texttt{eigenvectors.row}(i)^T ${{< /katex >}}
{{< alert context="warning" text="Assumes real eigenvalues." />}}

For detailed information, please refer to the OpenCV documentation

EigenVector in

output matrix of eigenvectors; it has the same size and type as src; the eigenvectors are stored as subsequent matrix rows, in the same order as the corresponding eigenvalues.
