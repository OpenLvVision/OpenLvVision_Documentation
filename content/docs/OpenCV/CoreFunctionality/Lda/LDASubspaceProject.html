OpenLvVision_OpenCv.lvlib:LDASubspaceProject.vim

Projects input data samples into a linear subspace defined by the Eigenvectors (W) and the Mean. This function is generic and can be used to project data using weights calculated by either PCA or LDA.

For detailed information, please refer to the OpenCV documentation
