Src

The training data matrix. The data must be floating-point (Single or Double). The layout is row-major, meaning each row represents one sample and each column represents a feature.

Labels

The labels corresponding to the training data. This must be a 1D array. The number of labels must exactly match the number of rows in the Data input. Example: [0, 0, 1, 1] indicates the first two samples belong to Class 0 and the next two belong to Class 1.
