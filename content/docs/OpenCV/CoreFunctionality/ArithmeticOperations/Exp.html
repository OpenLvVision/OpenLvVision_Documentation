Calculates the exponent of every array element.The maximum relative error is about 7e-6 for single-precision input and less than 1e-10 for double-precision input. Currently, the function converts denormalized values to zeros on output. Special values (NaN, Inf) are not handled.

The function exp calculates the exponent of every element of the input array:

{{< katex >}}$ \texttt{dst}[I] = e^{\texttt{src}(I)} ${{< /katex >}}

For detailed information, please refer to the OpenCV documentation
