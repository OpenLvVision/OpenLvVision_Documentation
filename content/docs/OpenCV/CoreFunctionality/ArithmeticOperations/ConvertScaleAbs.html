OpenLvVision_OpenCv.lvlib:ConvertScaleAbs.vim

Scales, calculates absolute values, and converts the result to 8-bit.

On each element of the input array, the function convertScaleAbs performs three operations sequentially: scaling, taking an absolute value, and conversion to an unsigned 8-bit type:

{{< katex >}}$ \texttt{dst}(I) = \texttt{saturate\_cast} \left( \left| \texttt{src}(I) \cdot \texttt{alpha} + \texttt{beta} \right| \right) ${{< /katex >}}

For detailed information, please refer to the OpenCV documentation
