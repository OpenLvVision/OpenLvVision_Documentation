OpenLvVision_OpenCv.lvlib:Randu.vim

Generates a single uniformly-distributed random number or an array of random numbers.

The non-template variant of the function fills the matrix dst with uniformly distributed random numbers from the specified range:

{{< katex >}}$ \texttt{low}_c \leq \texttt{dst}(I)_c < \texttt{high}_c ${{< /katex >}}

For detailed information, please refer to the OpenCV documentation

Dst

output array of random numbers; the array or image must be pre-allocated.

Low

Inclusive lower boundary of the generated random numbers.

Every element gets mapped to an channel:

0 = Blue/Real

1 = Green / Imaginary

2 = Red

3= Alpha

    		Low

High

Exclusive upper boundary of the generated random numbers.

One element per channel.

Every element gets mapped to an channel:

0 = Blue/Real

1 = Green / Imaginary

2 = Red

3= Alpha

    		High
