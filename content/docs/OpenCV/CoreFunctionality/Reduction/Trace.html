Returns the trace of a matrix.

The function trace returns the sum of the diagonal elements of the matrix mtx:

{{< katex >}}$ \mathrm{tr}(\texttt{mtx}) = \sum_i \texttt{mtx}(i,i) ${{< /katex >}}
