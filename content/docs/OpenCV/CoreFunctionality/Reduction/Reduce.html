Reduces a matrix to a vector.

The function reduce reduces the matrix to a vector by treating the matrix rows/columns as a set of 1D vectors and performing the specified operation on the vectors until a single row/column is obtained. For example, the function can be used to compute horizontal and vertical projections of a raster image. In case of REDUCE_MAX and REDUCE_MIN, the output image should have the same type as the source one. In case of REDUCE_SUM, REDUCE_SUM2 and REDUCE_AVG, the output may have a larger element bit-depth to preserve accuracy.

For detailed information, please refer to the OpenCV documentation

Src

input 2D matrix.

ReduceType

REDUCE_SUM =the output is the sum of all rows/columns of the matrix.

REDUCE_AVG = the output is the mean vector of all rows/columns of the matrix.

REDUCE_MAX = the output is the maximum (column/row-wise) of all rows/columns of the matrix.

REDUCE_MIN = the output is the minimum (column/row-wise) of all rows/columns of the matrix.

REDUCE_SUM2 =the output is the sum of all squared rows/columns of the matrix.

Dim

dimension index along which the matrix is reduced. 0 means that the matrix is reduced to a single row. 1 means that the matrix is reduced to a single column.
