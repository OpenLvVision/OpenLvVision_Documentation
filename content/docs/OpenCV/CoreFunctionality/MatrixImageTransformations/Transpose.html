Transposes a matrix.

The function cv::transpose transposes the matrix src:

{{< katex >}}$ \texttt{dst}(i,j) = \texttt{src}(j,i) ${{< /katex >}}

The function performs a reflection of the matrix elements across its main diagonal. Note that in the case of a square matrix, the operation can be performed in-place if the source and destination arrays are the same.

For detailed information, please refer to the OpenCV documentation
