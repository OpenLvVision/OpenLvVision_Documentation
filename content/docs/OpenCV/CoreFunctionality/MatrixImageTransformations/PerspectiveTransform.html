Performs the perspective matrix transformation of vectors.

The function cv::perspectiveTransform transforms every element of src by treating it as a 2D or 3D vector, in the following way:

{{< katex >}}$ (x, y, z) \rightarrow (x'/w, y'/w, z'/w) ${{< /katex >}}

where the intermediate 4D vector is calculated as:

{{< katex >}}$ (x', y', z', w') = \texttt{mat} \cdot \begin{bmatrix} x \\ y \\ z \\ 1 \end{bmatrix} ${{< /katex >}}

and the scale factor $w$ is defined as:

{{< katex >}}$ w = \begin{cases} w' & \text{if } w' \neq 0 \\ \infty & \text{otherwise} \end{cases} ${{< /katex >}}

In the case of 2D vector transformation $(x, y)$, the $z$ component is omitted and the matrix mat is typically $3 \times 3$ or $4 \times 4$. The function is commonly used for warping images or projecting 3D points onto a 2D plane with perspective distortion.

{{< alert context="warning" text="The function transforms a sparse set of 2D or 3D vectors. If you want to transform an image using perspective transformation, use warpPerspective . If you have an inverse problem, that is, you want to compute the most probable perspective transformation out of several pairs of corresponding points, you can use getPerspectiveTransform or findHomography . " />}}
