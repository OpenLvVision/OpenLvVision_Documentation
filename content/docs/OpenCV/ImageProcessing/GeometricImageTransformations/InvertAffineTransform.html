Inverts an affine transformation.

The function computes an inverse affine transformation represented by matrix M:

{{< katex >}}$ \begin{bmatrix} a_{11} & a_{12} & b_1 \\ a_{21} & a_{22} & b_2 \end{bmatrix} ${{< /katex >}}

The result is also a 2 \times 3 matrix of the same type as M.

For detailed information, please refer to the OpenCV documentation
